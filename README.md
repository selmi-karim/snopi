# Installation
git clone https://github.com/selmi-kerim/snopi

```
npm install
npm start
```
or

```
yarn install
yarn start
```


#### Create .env file and fill the env vars:

```
APIAI_TOKEN=
APIAI_SESSION_ID=some_unique_session_id
```
